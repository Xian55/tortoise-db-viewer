🐢 Embeddable powered tooltips

Include one script on any page and links to the Tortoise-WoW database gain Wowhead-style hover tooltips — no backend, no iframe. Add before </body>:

<script src="https://xian55.github.io/tortoise-db-viewer/embed/tw-power.js" defer></script>

Hover any link below — the tooltip is the full in-game card (stats, sources, set bonuses, spell effects), fetched from the public JSON API. Two link styles are supported: SPA query links (?item=123) and Open Graph stub links (/i/123).
